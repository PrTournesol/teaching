1. Avantages

  • Facilité, réutilisation, syntaxe facile

  • Mélange de réponses et de questions (si besoin)

  • Correction automatique

2. Install

http://home.gna.org/auto-qcm/

$ port install auto-multiple-choice

AMC est un logiciel libre pour machine à base de Linux .

Pour les Windowsiens ⇒ Machine virtuelle!

3.4. Etape 4 : Rédaction des questions

Exemple de fichier de base
# AMC-TXT source
PaperSize: A4
Lang: FR
Title: Mon premier questionnaire

Presentation: Veuillez répondre aux questions
ci-dessous du mieux que vous pouvez.

* Quelle est la capitale du Cameroun ? (1)
+ Yaoundé (2)
- Douala  (3)
- Kribi

** Parmi les nombres suivants, (4)
lesquels sont positifs ?
+ 2
- -2
+ 10
1 Une * ⇒ Une seule réponse
2 Un + ⇒ Bonne réponse
3 Un - ⇒ Mauvaise réponse
4 Deux * ⇒ Réponses multiples

3.7. Etape 7 : Traitement des Scans

On place les fichiers de scan .pdf dans le répertoire VotreProjet/scans.

On prépare AMC au traitement des copies (bouton Calculer les mises en page )

On va dans l’onglet Saisie et on sélectionne le bouton Automatique .

5. Conclusion

  • Très très utile pour les grosses promos

  • Très complet

  • Demande un peu de pratique

  • Pas de compatibilité avec les QCM Moodle

  • A déconseiller aux allergiques à l’ordinateur
